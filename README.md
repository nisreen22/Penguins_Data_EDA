# 🐧 Palmer Penguins: Species & Biometric Analysis

## 📌 Project Overview
This project focuses on **Data Integration** and **Exploratory Analysis**. It merges multiple datasets representing different penguin species (Adelie, Gentoo, and Chinstrap) to analyze their distribution across islands and identify unique biometric features.

## 📊 Key Insights
- **Island Distribution:** Adelie penguins are found on all three islands, while Gentoo and Chinstrap are island-specific.
- **Biometric Clusters:** Clear correlations were found between Culmen Length and Culmen Depth, acting as distinct identifiers for each species.
- **Data Engineering:** Successfully integrated multiple CSV files into a unified clean dataset for analysis.

## 🛠️ Tools & Libraries
- **Data Processing:** Pandas (Concatenation, Cleaning).
- **Visualizations:** Created interactive visuals and charts using **Plotly Express** and **Seaborn** for dynamic data exploration.
## 🚀 How to Run
1. Ensure you have the three source CSV files in the same directory.
2. Run `Penguins.ipynb` to see the integration and visualization steps.
